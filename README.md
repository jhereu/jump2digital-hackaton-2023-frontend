## How to build

### Prerequisites

- [NodeJS](https://nodejs.org/en)

### Install

```sh
# Using npm
npm install

# Using yarn
yarn install
```

### Build

```sh
# Using npm
npm run build

# Using yarn
yarn build
```

### Run

```sh
# Using npm
npm run start

# Using yarn
yarn start
```

Then visit http://127.0.0.1:80/
